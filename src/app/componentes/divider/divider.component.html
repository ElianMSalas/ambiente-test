Descripción

El Divider es un componente visual que funciona como una línea divisoria para separar contenido dentro de una interfaz.

Uso del componente

El Divider se emplea en contextos donde es necesario segmentar visualmente diferentes secciones de contenido relacionadas sin establecer una separación semántica fuerte.

Diseño y Formato del componente

El Divider es una línea horizontal o vertical que se puede personalizar en grosor, color y longitud.

Criterios de accesibilidad

Criterios Transversales

  1. Contraste: El Divider debe ser visible sin afectar la jerarquía visual del contenido y debe mantener un contraste suficiente con el fondo para asegurar su percepción sin dificultar la lectura.

  2. Navegación por teclado: Al ser un elemento visual no interactivo, no debe interferir con la navegación por teclado.

  3. Atributos ARIA: Usar role="separator" para que los lectores de pantalla lo identifiquen correctamente cuando la separación tenga significado en la estructura del contenido.

  4. Compatibilidad con lectores de pantalla: Si el Divider es puramente decorativo, se debe ocultar con aria-hidden="true" para evitar distracciones innecesarias en tecnologías de asistencia.

Criterios Específicos

  1. Uso adecuado: No emplear el Divider como sustituto de encabezados, títulos o espacios en blanco.

  2. Contexto: Si la separación de contenido es clave para la comprensión o navegación, combinarlo con h2, section, o aria-labelledby para proporcionar contexto adecuado.
