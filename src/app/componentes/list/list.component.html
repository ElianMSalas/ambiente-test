List (Lista)

Descripción

Una lista HTML es un componente fundamental en el desarrollo web que permite organizar información de manera estructurada y fácil de leer. Se utiliza para presentar elementos relacionados de forma secuencial o no, lo que facilita la comprensión y la navegación para los usuarios.

Uso del componente

El componente lista se utiliza para mostrar un conjunto de artículos o elementos que deben ser organizados de manera estructurada y clara. Estos elementos se usan principalmente para mostrar información que agrupa categorías, lista de productos, tareas, enlace u otros ítems.

Diseño y Formato del componente

Este tipo de componente contiene elementos los cual se ordenan de forma natural verticalmente. Angular Material no utiliza el decorativo nativo de HTML, dejándolo como (list-style-type: none;).

Criterios de accesibilidad

Criterios Transversales

  1. Contraste: Los elementos dentro de una lista deben tener suficiente contraste de texto.

  2. Navegación por teclado: Las listas deben ser navegables con Tab y las opciones deben ser seleccionables con las teclas de flecha.

  3. Atributos ARIA: Usar role="list" y role="listitem" para definir la estructura semántica de la lista.

  4. Compatibilidad con lectores de pantalla: El lector de pantalla debe identificar la lista y leer cada ítem de la lista con aria-live si se actualiza dinámicamente.

Criterios Específicos

  1. Listas con encabezado o grupo de elementos: Asociar cada grupo con un encabezado mediante aria-labelledby para mejorar la comprensión jerárquica.

  2. Listas interactivas (con botones o enlaces dentro):

    • Asegurar que cada elemento de la lista sea un botón (button) o un enlace (a href="#">) si es clickeable.

    • Usar aria-pressed="true/false" si el elemento representa un estado alternable.

  3. Listas dinámicas o cargadas asincrónicamente: Implementar aria-live="polite" si la lista se actualiza con nuevos elementos en tiempo real.

  4. Listas con iconos o imágenes: Proporcionar alt en imágenes dentro de listas o usar aria-hidden="true" si son decorativas.

  5. Etiquetado HTML Semántico: Implementar listas con ul y li correctamente.
