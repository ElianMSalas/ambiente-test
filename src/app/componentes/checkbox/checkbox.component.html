Descripción

El componente Checkbox, o casilla de verificación es un selector múltiple de elementos.

Uso del componente

Permiten seleccionar uno o varios elementos de una lista o tabla según sea necesario. Esto facilita la ejecución de acciones en grupo, como eliminar elementos o continuar un proceso con la selección realizada. Además, pueden tener un funcionamiento dependiente, donde un checkbox principal controla a varios secundarios, permitiendo seleccionar o deseleccionar todos los elementos con un solo click.

Diseño y Formato del componente

El diseño de los checkboxes suele ser cuadrado, con un fondo transparente y bordes sutiles cuando están deseleccionados. Al seleccionarlos, el fondo adquiere color y se muestra un ícono de check para indicar la elección.

Criterios de accesibilidad

Criterios Transversales

  1. Contraste deben cumplir con los requisitos de contraste según WCAG 2.1 AA.

  2. Navegación por teclado: Debe ser seleccionable con Space.

  3. Atributos ARIA: El Checkbox debe ser descriptivo por si mismo. Sin embargo, si necesitas un contexto adicional es recomendable utilizar atributos ARIA.

  4. Compatibilidad con lectores de pantalla: Los lectores de pantalla deben anunciar el estado de la casilla usando aria-checked="true/false".

Criterios Específicos

  1. Gestión de estado: Una vez seleccionada la opción de checkbox, debe ser fácilmente identificatoria, respetando el diseño y formato del componente.

  2. Etiquetado HTML Semántico:

    • Recuerda que para los inputs se debe utilizar la etiqueta input type="checkbox"
