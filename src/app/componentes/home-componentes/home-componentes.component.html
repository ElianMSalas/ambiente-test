Componentes

Los componentes son elementos reutilizables de la interfaz de usuario que facilitan la construcción de sistemas digitales coherentes y accesibles. Cada componente tiene un propósito específico y una funcionalidad definida, ayudando a estructurar y mejorar la experiencia de las personas usuarias.

Autocomplete

Campo de entrada que sugiere opciones a medida que la persona usuaria escribe.

Badges

Indicadores pequeños para mostrar notificaciones, conteos o estados sobre un elemento.

Breadcrumbs

Muestra la ruta de navegación para facilitar la orientación dentro del sitio.

Buttons

Elementos interactivos para ejecutar acciones en la interfaz.

Buttons Toggle

Botones agrupables que permiten alternar entre opciones.

Card

Contenedor visual para agrupar información de manera estructurada.

Checkbox

Permite seleccionar u o múltiples opciones en formularios.

Chips

Elementos compactos que representan entradas, categorías o etiquetas.

Datepicker

Permite seleccionar fechas o periodos de un calendario.

Dialog

Ventana emergente para mostrar información o solicitar interacción.

Divider

Línea visual que separa contenido dentro de una interfaz.

Expansion Panel

Contenedor colapsable para mostrar u ocultar contenido adicional.

Input

Campo de entrada para capturar datos en formularios.

Links

Enlaces que permiten la navegación entre secciones o sitios externos.

List

Agrupación estructurada de elementos relacionados en la interfaz.

Menu

Lista desplegable de opciones o acciones dentro de un botón.

Paginator

Control de paginación para navegar entre grandes volúmenes de datos.

Progress Bar

Indicador visual de progreso en procesos en curso.

Progress Spinner

Animación circular para representar carga o espera.

Radio button

Permite seleccionar solo una opción dentro de un grupo.

Select

Menú desplegable para elegir una o varias opciones predefinidas.

Sidenav

Panel lateral deslizante para navegación o contenido adicional.

Slide Toggle

Interruptor visual para activar o desactivar opciones.

Slider

Control deslizante para seleccionar un valor dentro de un rango.

Snackbar

Mensaje emergente breve para notificaciones temporales.

Table

Presentación estructurada de datos en formato tabular.

Tabs

Navegación entre secciones dentro de una misma interfaz.

Tooltip

Mensaje emergente con información adicional al pasar el cursor.
